# AI Powered YouTube Analytics

A Machine Learning based web application that predicts YouTube video views using XGBoost.

## Features

- 📊 ML Prediction
- ⚡ FastAPI Backend
- 🎨 Responsive UI
- 📈 Dashboard
- 📂 Excel Dataset
- 📉 Charts

## Technologies

- Python
- FastAPI
- XGBoost
- HTML
- CSS
- JavaScript
- Chart.js

## Run

```bash
pip install -r backend/requirements.txt

python backend/train_model.py

python -m uvicorn backend.app:app --reload
```

Open:

```
http://127.0.0.1:8000
```
